## Thread 4 — Prompt Framework Tutorial (RSCF)

**Tweet 1 (hook):**
```
Most people use ChatGPT wrong.

Here's the prompt framework that changed my results: 🧵
```

**Tweet 2:**
```
The problem: vague prompts = vague outputs.

"Write me a blog post about AI tools" gets you something generic enough to be useless. The AI isn't the bottleneck. Your prompt is.
```

**Tweet 3:**
```
The fix: RSCF framework.

→ Role
→ Situation
→ Constraints
→ Format

Four elements. Takes 60 extra seconds to write. The output quality difference is not subtle.
```

**Tweet 4:**
```
Role: Tell the AI who to be.

"You are a senior B2B content strategist with 10 years of experience in SaaS marketing."

This sets tone, vocabulary, and depth automatically. Don't skip it.
```

**Tweet 5:**
```
Situation: Give context.

"I'm writing for a technical audience of CTOs who are skeptical of AI hype and have seen too many failed implementations."

The more specific your situation, the more targeted the output.
```

**Tweet 6:**
```
Constraints: Set guardrails.

"No buzzwords. No more than 800 words. Use specific examples and data, not generalities. Avoid phrases like 'in today's fast-paced world.'"

Constraints force quality. Unconstrained = generic.
```

**Tweet 7:**
```
Format: Specify the output structure.

"Return a structured outline with H2s first. Once I approve the outline, write the full draft."

This catches direction problems before you're 800 words in.
```

**Tweet 8:**
```
Combined example:

"You are a senior B2B content strategist. I need a post for skeptical CTOs — max 800 words, no buzzwords, specific examples only. Give me an outline first, then the full draft."

That's it. Try it.
```
